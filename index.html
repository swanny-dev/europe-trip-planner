Quick guide

How this planner works

1

Choose your role

Use the Ruairi or Maggie link. Each person can only press their own vote button, so the yes votes stay honest.

2

Add destinations

Use Destinations to add countries or cities, set nights, and add ideas. New places get starter ideas and a representative image.

3

Build the route

The Overview tab is the working route. Add stops, reorder them, leave notes, and open the date chip when you want to set a range.

4

Keep the money real

The Budget tab estimates the trip from flights, daily costs, nights, mutual yes ideas, and a buffer.

5

Shared saving

When the status says Saved online, changes are stored on the shared planner so the other person can refresh and see them.
